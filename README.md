# コンソールRPG（C++）

## 概要
C++で作成したコンソールベースのRPGゲームです。  
マップ移動、ランダムエンカウント、戦闘システム、パッシブスキル獲得・強化などを実装しています。
学生時代に学習したC++の復習からC++の基礎文法、クラス設計、継承、仮想関数、STLの使い方を学ぶ目的で制作しました。

---

## ゲーム内容
- マップ上を移動しながら探索
- ランダムマスで敵とエンカウント
- 戦闘中に攻撃・防御を選択
- 特定のマスでパッシブスキルを獲得
- ボスを倒すとゲームクリア

---

## 実装済み機能

### マップ関連
- 二次元配列によるマップ管理
- プレイヤー移動処理
- イベントマス判定  
  - ランダムマス  
  - 回復マス  
  - スキル獲得マス  
  - ボスマス

### 戦闘システム
- ターン制バトル
- 攻撃 / 防御コマンド
- 防御時の被ダメージ軽減
- 敵・プレイヤーのHP管理
- 敵のランダム出現

### キャラクター設計
- `Character` クラスを基底とした  
  - `Player`
  - `Enemy`
- 継承による共通処理の再利用
- 仮想関数によるダメージ処理の分離

### スキルシステム
- `Skill` 抽象クラス（純粋仮想関数 `apply`）
- スキルを `vector<Skill*>` で管理
- スキル獲得時の重複判定とレベルアップ
- 戦闘中一度だけ発動する条件付きパッシブスキル
  - HP80%以下で攻撃力上昇
  - HP30%以下で大幅攻撃力上昇

### 使用技術
- C++
- STL（`vector`）
- 乱数生成（`<random>`）
- クラス設計 / 継承 / 仮想関数
- 状態管理（マップ / 戦闘 / イベント）
