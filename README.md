# コンソールRPG(C++)

## 概要
C++で作成したコンソールベースのRPGゲームです。
マップ移動、ランダムエンカウント、戦闘システム、パッシブスキル獲得・強化などを実装しています。

---

## ゲーム内容
-マップ上を移動しながら探索
-ランダムマスで敵とエンカウント
-戦闘中に攻撃・防御を選択
-戦闘終了後に弱パッシブスキル、特定のマスで強パッシブスキルを獲得
-ボスを倒すとゲームクリア

---

## 実装済み機能

### マップ関連
-二次元配列によるマップ管理
-プレイヤー移動処理
-イベントマス判定
  -ランダムマス
  -回復マス
  -スキル獲得マス
  -ボスマス

### 戦闘システム
-ターン制バトル
-攻撃 / 防御コマンド
-防御時の被ダメージ軽減
-敵・プレイヤーのHP管理
-敵のランダム出現

### キャラクター設計
- `Character' クラスを基底とした
  - `選手`
  - `敵`
-継承による共通処理の再利用
-仮想関数によるダメージ処理の分離

### スキルシステム
- `Skill' 抽象クラス(純粋仮想関数「apply」)
-スキルを 'vector<Skill*>' で管理
-スキル獲得時の重複判定とレベルアップ
-戦闘中に発動する条件付きパッシブスキル

### 使用技術
-C++
-STL(「ベクトル」)
-乱数生成('<random>')
-クラス設計 / 継承 / 仮想関数
-状態管理（マップ / 戦闘 / イベント）

## ビルド・実行方法

### 推奨環境
-窓
-Visual Studio(C++ 開発環境)

### 手順
1.Visual Studioで新規のC++コンソールプロジェクトを作成
2. 本リポジトリ内の '.cpp' / '.h' ファイルをすべてプロジェクトに追加
3. ビルドして実行

※ Visual Studioの自動生成ファイル(.vs / Debug / Release など)は含めていません。

---

## 工夫した点
-処理ごとにクラスを分離し、役割を明確化
-スキル処理を仮想関数で統一し、拡張しやすい構成にした
-戦闘処理とマップ処理を分離して可読性を意識
-理解しやすい構造を目指した
